# Форк PawTotems (версия для мода Hold My Items)
Продолжение ресурспака PawTotems с нововведениями и улучшениями. Оригинальным создателем является [Spilya](https://github.com/Spilya/PawTotems/).

PawTotems добавляет тотемы с игроками (преимущественно с #СП). Отличительной особенностью является поддержка [Ears Mod](https://ears.y2k.diy/).

Нововведения по сравнению с оригинальным ресурспаком:
1. **Отсутствие пустых обновлений.** Оригинальный PawTotems имел привычку обновлять номер сборки по 100500 раз в сутки. Этот так не делает — смена номера происходит только в случае действительного обновления ресурспака. Сам он обновляется по 2-3 раза в неделю в 3:00 по МСК.
2. **Поддержка игроков со скинами без Ears Mod функционала.**
3. **В пак может добавиться любой игрок.** Раньше некоторые игроки не могли быть добавлены по субъективным причинам. **Я принимаю любого игрока.**
4. **Исправление MC-169950.** В реализации PawTotems от Spilya текстура хвоста имела размер **32x24**, что ломало мип-маппинг. У меня все текстуры хвоста размером **32x32**, которые не вызывают проблем.
5. **Исправленное положение тотема в руке с модом Hold My Items.**

Написан код сборки PawTotems был **с нуля** из-за отсутствия исходников.

PawTotems поддерживает обновления через [DynamicPack](https://modrinth.com/mod/dynamicpack).
